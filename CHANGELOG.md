# Changelog

All notable changes to Drift will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] — 2026-03-16

### Added
- **Core jiggling** with three modes: Simple, Zen, and Human
- **Keep Awake** via `chrome.power.requestKeepAwake("display")`
- **Keyboard shortcut** `Alt+Shift+D` to toggle on/off
- **Badge indicator**: green (active), red (inactive), yellow (scheduled)
- **Smart Scheduling**: work hours, lunch break, active days
- **Quick timers**: 30m, 1h, 2h, 4h one-click activation
- **Break reminders**: 20-20-20 eye rule, stretch reminders (configurable)
- **Presence detection**: detect Microsoft Teams and Slack in open tabs
- **Human-like activity patterns**: variable speed, micro-movements, pauses, scrolls, keystrokes
- **Popup UI**: dark theme, toggle, mode selector, timer, schedule, presence
- **Settings page**: jiggle config, schedule, breaks, theme, notifications
- **Welcome page**: 3-slide onboarding carousel on first install
- **Theme system**: dark (default), light, system auto
- **i18n**: English, Spanish, Portuguese
- **Privacy-first**: zero data collection, all local storage
- **Chrome Web Store assets**: description, privacy policy
- **Documentation**: architecture docs, test plan
